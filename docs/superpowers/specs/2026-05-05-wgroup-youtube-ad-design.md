# WGroup YouTube Ad Film — Design Spec

**Date:** 2026-05-05
**Owner:** berkeguleryuz@gmail.com
**Status:** Approved (pending user review of this written spec)

## 1. Goal

Produce a 30-second horizontal (16:9) YouTube ad film that introduces WGroup, communicates its tagline ("The Art of Transforming Knowledge into Excellence"), and showcases its three divisions (W-Quality, W-DigiLab, W-Studio) through a tagline-driven montage: **Knowledge → Transform → Excellence**.

Image generation: **Higgsfield MCP** with the **gpt_image_2** model only.
Video generation: **Higgsfield MCP** image-to-video only, using start-frame + end-frame chaining so consecutive scenes flow seamlessly.
Voiceover: **OpenAI TTS** (`gpt-4o-mini-tts`, voice `nova`), single female voice, full English.
Background music: **YouTube Audio Library**, cinematic/inspirational, very low mix.
All intermediate outputs (images, video clips, audio takes) are saved in versioned per-scene folders so any single piece can be regenerated or replaced without rebuilding the whole film.

## 2. Constraints

- Total duration: **30 seconds** (target 28–30s).
- Aspect ratio: **16:9 horizontal**, 1920×1080 master.
- Language: **English only**.
- All visuals must be generated by **Higgsfield MCP gpt_image_2**. No other image source.
- All video clips must be generated by **Higgsfield MCP image-to-video** with explicit start frame and end frame.
- Three category images are mandatory and must appear in the film:
  - `public/w-new/W-Quality.webp` → start frame of S2
  - `public/w-new/W-Digilab.webp` → start frame of S3
  - `public/w-new/W-Studio.webp` → start frame of S4
- These three images additionally serve as **style/mood references** passed to gpt_image_2 when generating other frames, to keep look consistent.
- Logo: `public/wgroup/logo.png` appears in S6 (logo reveal).
- Aesthetic: **hybrid documentary realism + subtle tech overlay** (HUD, data lines, scan elements). Bright, optimistic, dawn/golden-hour lighting — never dark.
- Single female voiceover (`nova`).
- Background music: cinematic, instrumental, ~80–90 BPM, no vocals, license-free, mixed very low (-22 to -26 LUFS under voice).

## 3. Story Arc

Tagline-driven montage. Each division gets one beat tied to one keyword:

| Beat | Keyword | Division |
|------|---------|----------|
| Open | World transforming | — |
| 1 | **Knowledge** | W-Quality |
| 2 | **Transform** | W-DigiLab |
| 3 | **Excellence** | W-Studio |
| Converge | Three → One | — |
| Close | Logo + URL | — |

## 4. Scene Plan

Each scene is ~5 seconds. Start frame of scene N = end frame of scene N-1 (seamless chain). Mandatory category images sit at the boundaries.

### S1 — Opening: World & Digital Transformation (0:00–0:04)

- **Look:** Earth from low orbit at golden-hour / dawn. Bright blue atmospheric glow, white cloud sheets, continents lit warmly. Gold-and-white data ribbons trace between continents (light, not menacing). Subtle HUD scan lines drift across.
- **Camera:** Slow dolly-in from orbit toward Europe.
- **Start frame:** Earth from orbit, full disc visible, Europe centered, data ribbons forming.
- **End frame:** Camera has descended; composition transitions into the W-Quality.webp framing (so S2 starts there cleanly).
- **VO:** *"In a world rebuilt by transformation,"*

### S2 — KNOWLEDGE / W-Quality (0:04–0:09)

- **Start frame:** `public/w-new/W-Quality.webp` (mandatory, used literally).
- **Look:** Industrial inspection / consulting environment, precision instruments, blueprints, IATF 16949 cues, soft daylight. Tech overlay: measurement HUD, data callouts, certification stamps fading in.
- **Camera:** Slow push-in revealing detail.
- **End frame:** Detail dissolves into a digital data plane — bridges to S3 (DigiLab).
- **VO:** *"excellence begins with knowledge — measured, proven, certified."*

### S3 — TRANSFORM / W-DigiLab (0:09–0:14)

- **Start frame:** `public/w-new/W-Digilab.webp` (mandatory, used literally).
- **Look:** Digital lab — code surfaces, dashboards, digital-twin renderings, particle data flows in WGroup blue/red. Bright, clean, optimistic.
- **Camera:** Lateral glide; data streams sweep through frame.
- **End frame:** Data converges into a single beam of light; transitions toward a media/storytelling space (S4).
- **VO:** *"It evolves through transformation — where data becomes decision."*

### S4 — EXCELLENCE / W-Studio (0:14–0:19)

- **Start frame:** `public/w-new/W-Studio.webp` (mandatory, used literally).
- **Look:** Cinematic studio environment — cameras, lighting, screens displaying industrial storytelling, warm key light. Tech overlay subdued.
- **Camera:** Crane-down or pull-back revealing the studio scale.
- **End frame:** Three light streams (red, deep blue, light blue — logo palette) converge in mid-air at frame center.
- **VO:** *"And it speaks through stories that move industries forward."*

### S5 — Three Converge (0:19–0:24)

- **Start frame:** Three converging streams from S4 end.
- **Look:** Streams weave into a luminous ribbon shape that begins to take the W-Group "W" silhouette. Bright, ethereal, particles, golden core.
- **Camera:** Static; subject animates.
- **End frame:** Ribbon resolves into the WGroup logo silhouette on a near-white background.
- **VO:** *"Three divisions. One vision. One signature of excellence."*

### S6 — Logo Reveal & CTA (0:24–0:30)

- **Start frame:** Logo silhouette from S5 end.
- **Look:** Final logo render from `public/wgroup/logo.png` lands center. Below it: *"The Art of Transforming Knowledge into Excellence"*. Bottom: `wgroupgmbh.eu`. Subtle particle settle, light bloom.
- **Camera:** Static, gentle ambient drift.
- **End frame:** Logo + tagline + URL hold.
- **VO:** *"WGroup — the art of transforming knowledge into excellence."*

## 5. Voiceover Script

Voice: OpenAI `gpt-4o-mini-tts`, voice `nova`. Tone: confident, modern, warm, premium-tech. Pace: moderate. Render at 24 kHz, normalize to -16 LUFS.

```
[S1 0:00] In a world rebuilt by transformation,
[S2 0:04] excellence begins with knowledge — measured, proven, certified.
[S3 0:09] It evolves through transformation — where data becomes decision.
[S4 0:14] And it speaks through stories that move industries forward.
[S5 0:19] Three divisions. One vision. One signature of excellence.
[S6 0:24] WGroup — the art of transforming knowledge into excellence.
```

Total ~71 words. Fits ~28s with natural pauses.

## 6. Audio Bed

- Source: **YouTube Audio Library** → Cinematic / Inspirational.
- Mood: optimistic, slow build, instrumental, ~80–90 BPM, no vocals.
- Mix: -22 to -26 LUFS (well under voice).
- Soft sub-rumble + light synth pad. No heavy drums.
- Final master loudness target: **-14 LUFS** (YouTube standard).


## 8. Production Pipeline

1. **Setup**: create `public/ad-render/` tree; initialize `manifest.json`.
2. **Reference upload**: upload the 3 category images + logo to Higgsfield as references.
3. **Frame generation (gpt_image_2)** per scene:
   - Build start frame (skip for S2/S3/S4 — those use the .webp directly).
   - Build end frame, passing prior scene's end frame and the relevant category image as references for style coherence.
   - Save every variant; update `manifest.json` to point to the chosen version.
4. **Video generation (Higgsfield image-to-video)** per scene:
   - Pass start frame + end frame + motion prompt.
   - Save every clip variant.
5. **Voiceover**: render full script with OpenAI TTS `nova`; also render per-line files for granular replacement.
6. **BGM**: pull cinematic/inspirational track from YouTube Audio Library; trim/loop to 30s.
7. **Compositing** (ffmpeg): concatenate clips, mix audio bed under voice, add subtle crossfades at scene boundaries (8–12 frames), overlay logo/URL/tagline text in S6, export 1920×1080 H.264 MP4 at -14 LUFS.
8. **QA**: review master, mark per-scene replacement candidates if any.

## 9. Acceptance Criteria

- 16:9, 1920×1080, ~30s, H.264 MP4.
- All three category images appear as start frames of S2/S3/S4.
- Logo appears clearly in S6.
- VO is intelligible at -22 LUFS bed; no clipping; -14 LUFS master.
- Scene transitions are smooth (start/end frame chain visible).
- Bright, optimistic look throughout — no dark/dystopian frames.
- Every intermediate asset present in `public/ad-render/` for selective replacement.

## 10. Open Items / Risks

- gpt_image_2 may need multiple attempts to keep brand color palette (deep blue, red, light blue) consistent across scenes — mitigated by passing previous scene's end frame as reference.
- Higgsfield image-to-video preserving exact end frame is best-effort; if drift is visible at scene boundary, we add a 6-frame crossfade as fallback.
- BGM track selection requires user listen-and-pick before final mix.
- ElevenLabs alternative remains available if `nova` quality is insufficient on first render — escalation path, not default.
